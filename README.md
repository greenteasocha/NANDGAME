# NANDGAME

Nandgameの内容をC++で再現する  
https://nandgame.com/

## ルール
内部に処理を書いてよいのはNAND演算子(素子？)のみ  
その他の演算子は  
 - コンストラクタによる入力のつなぎ合わせ  
 - すでに作成した演算子による倫理演算、数値演算

以外を禁止  
(定義がないので個人の良心に任せられるが、頑張る)
